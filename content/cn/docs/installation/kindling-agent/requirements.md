### Dependency

| **Dependency**                     | **Description**                  |
| ---------------------------------- | -------------------------------- |
| Kubernetes                         | Kubernetes v1.x+ is required && not supported Minikube   |
| Prometheus and Prometheus Operator | Prometheus is already installed. |

### Architecture

| **architecture** | **Support**   |
| ---------------- | ------------- |
| x86-64           | Supported     |
| ARM              | Not supported |
